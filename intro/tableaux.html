Les listes à puces

Les listes à puces peuvent être ordonnées ou non ordonnées
Les listes à puces ne doivent pas être encapsulée dans des paragraphes (p)

Listes non ordonnées

  • Introduction
  • Premier point
  • Deuxième point
    • premier sous point
      • Sous sous point 1
      • Sous sous point 2
  • Troisième point
  • Conclusion

Liste ordonnée

  1. Intro
  2. Partie
    1. point
    2. point
  3. Point
    1. point
    2. point

Liste ordonnée inversée

  1. Intro
  2. Partie
    1. point
    2. point
  3. Point
